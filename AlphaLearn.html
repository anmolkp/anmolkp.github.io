ALPHALEARN AR


PURPOSE OF BUILDING

AlphaLearn AR is an Educational Augmented Reality Application created for the Android Platform.

The application was built to help specially-abled children learn alphabets in a more visual manner.

PROJECT DETAILS

AlphaLearn AR was created using the Unity 3D Game Engine and the Vuforia SDK. It was programmed in the C# language.

The project has been configured to work with Android Phones.

The user points the camera to a page of a particular book and a 3-D model of the alphabet appears on the page of that book.

For example, if you point the camera to a page filled with pictures of an ant and an apple, it will show a 3-D model of the letter A.

The project is currently built to work with just one particular book for alphabets.
